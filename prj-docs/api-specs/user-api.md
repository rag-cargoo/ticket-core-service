# 👤 User API Specification

티켓 서비스 이용을 위한 사용자 프로필을 관리하는 API입니다.

---

## 🎯 1. 기능 개요
- 사용자는 본인의 고유한 `username`으로 시스템에 등록되어야만 예매 서비스에 참여할 수 있습니다.
- 유저 ID는 시스템 전반(공연 조회, 예약 시도 등)에서 필수 식별자로 사용됩니다.

---

## 🛠️ 2. API 상세 명세

### 2.1. 신규 유저 생성 (Sign-up)
- **기능**: 새로운 티켓 예매 권한을 가진 사용자를 생성합니다.
- **제약 사항**: 동일한 `username`은 중복 등록이 불가능합니다. (409 Conflict 또는 에러 발생)

### 2.2. 유저 정보 조회
- **기능**: ID를 기반으로 사용자의 프로필 정보를 가져옵니다.

### 2.3. 유저 계정 삭제
- **기능**: 시스템에서 해당 유저의 모든 정보를 제거합니다.
- **🚨 주의 (Business Rule)**: 해당 유저에게 **'진행 중'이거나 '확정된' 예약**이 있을 경우, 데이터 무결성을 위해 삭제가 제한될 수 있습니다. 반드시 예약 취소를 먼저 진행해야 합니다.
